在Windows上使用Vim问题就是多，又遇到编码问题，蒙哥阅读器导出的单词本文件在Linux上打开是显示正常，在Windows上打开显示成乱码，只是中文乱码，英文没有问题，解决办法如下：


1. 在```_vimrc```中添加如下语句

   ```vimscript
   set encoding=utf-8
   set fileencodings=utf-8,gbk,cp936,gb18030,gb2312,ucs-bom,latin-1
   ```

2. 用```:set fileconding```查看文件编码格式
