** 全局上下文

在全局执行环境中(在任何函数体外部)this指向
**全局对象**

在浏览器中是windows对象

**全局对象可以用globalThis获取**

** 函数上下文

在函数内部,this的值取决于函数被调用的方式.

#+begin_src js
function f1(){
  return this;
}
#+end_src

- 直接调用

  在浏览器中直接调用f1(),会返回window对象.
  
  在严格模式下如果没有绑定this,在函数内部调用this是undefied.

- 作为对象的属性或方法调用

  会返回window对象.
